Frontend (Angular) - scaffolding mínimo

Pasos rápidos:

1) Ir a `frontend/` e instalar dependencias:

```powershell
npm install
```

2) Para desarrollo con Angular CLI (si deseas generar el app completo):

```powershell
npm install -g @angular/cli
ng new . --routing --style=css
```

3) Ejecutar (si usas Angular CLI):

```powershell
npm start
```

4) Ejecutar tests con Jest:

```powershell
npm test
```

Archivos generados: `src/app/components`, `src/app/services`, `views/reportes` y un test de ejemplo con Jest.
